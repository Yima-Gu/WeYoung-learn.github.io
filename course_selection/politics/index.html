    思政篇

    思想道德与法治

    朱效梅老师

    • 课程介绍： 未提供详细课程介绍。
    • 课程任务： 两篇读书报告和一次小组pre。
    • 评价： 评价存在一些矛盾：有同学反映老师给分很好，但也有同学认为任务量较大、给分普通。普遍认为小组pre比较“卷”，竞争激烈。
    • 使用教材： 无

    张瑜老师

    • 课程介绍： 老师考核方式多样，任务较为零碎，包含线上慕课、课堂讲授和小班研讨等多种形式。
    • 课程任务： 任务量大，包括：每周MOOC及答题、随机点名、2-3篇报告（如读书报告、成才报告）、两次小班研讨/pre以及期末开卷考试。
    • 评价： 对任务量的体感普遍是“大”或“零碎”。给分评价不一，早期评论认为认真完成则给分不错，但近年有评论指出给分变差，且期末考试占比较重。总体上，认真对待各项任务有机会获得高分。
    • 使用教材： 《思想道德与法治》

    吴俊老师

    • 课程介绍： 课程压力不大，以一个有特色的小组实践任务为核心。
    • 课程任务： 有一个贯穿前几周的小组大作业（如“社会主义核心价值观”主题实践），小组成员需合作完成报告或视频。课堂压力小，作业不多，可能没有pre。
    • 评价： 老师人很好，给分不错。同学普遍反映任务量不算大，积极参与小组活动和讨论容易取得较好成绩（A/A-），是推荐选择的老师。
    • 使用教材： 《思想道德与法治》

    李江静老师

    • 课程介绍： 老师“人美心善”，课程质量高，能给同学带来思想上的启发。上课形式为讲授与小组pre交替进行，需要签到。
    • 课程任务： 无期末考试。任务包括：一篇读书笔记、一篇专题报告以及一次小组pre。
    • 评价： 学生评价非常高。认为老师和善，课程有深度，任务量适中。是一门能有收获的高质量思政课。
    • 使用教材： 无

    邓喆老师

    • 课程介绍： 年轻老师，上课会组织讨论，但有随机提问，不参与会被记录。老师个人风格较为突出，在学生中评价两极分化严重。
    • 课程任务： 任务量不少，包括一次竞争非常激烈的小组pre（有助教指导和同学投票环节）、若干有字数上限的小文章写作。
    • 评价： 评价非常两极化。有同学认为老师温柔、给分好；但近年有大量同学给出“避雷”警告，指出老师存在强制分组、对待学生不公、强制参与等问题，体验“恶心”。也有同学认为没有传闻中那么差，但承认pre非常“卷”。选择该老师风险较大。
    • 使用教材： 《思想道德与法治》

    肖巍老师

    • 课程介绍： 老师研究哲学和伦理学，课堂内容也多围绕相关议题（如性别平等、AI伦理等）展开，讨论氛围开放包容，基本不使用课本。课程模式有过多种变化，如“大班授课+小班讨论”或“讲授+翻转课堂”。
    • 课程任务： 任务量和形式随年份有变化。通常包括2-4次几百到一千多字的小论文/读书报告，有时有小组pre，期末为开卷论述题或大论文。
    • 评价： 评价不一且随时间变化。早期评价多为“任务量轻”、“适合划水”、“给分还行”。近期评价出现分歧，有同学认为pre很卷、给分差（即使认真完成也只有3.3）；但也有同学认为课堂包容性强、pre表现好能拿4.0。总体而言，课程内容有趣，但近年给分似乎有不确定性。
    • 使用教材： 《思想道德与法治》（但老师上课基本不用）

    王雯殊老师

    • 课程介绍： 老师讲课认真，会加入自己的见解，需要记笔记。每节课会点名。前几周可能只需参加一次小班讨论课。
    • 课程任务： 任务量不大。包括一次小班讨论/辩论、一篇读书报告、两次小作业和期末开卷考试。考试会涉及少量老师上课强调的书本外内容。
    • 评价： 老师讲课受认可，但给分被普遍评价为“玄学”，没有清晰的给分依据。有认真完成所有任务仅得3.3的，也有运气好拿到4.0的。同学认为文字功底好可能占优。适合不在意绩点、追求任务轻松的同学。
    • 使用教材： 《思想道德与法治》

    戴木才老师

    • 课程介绍： 老师被评价为很“佛系”，但上课内容普遍认为比较无聊，有时会用雨课堂点名。
    • 课程任务： 任务量轻，没有pre。一般是两次1500字左右的小论文，可能有MOOC和小班讨论。期末为开卷考试，且会提前给出思考题范围，考试题目为原题。
    • 评价： 经典的“水课”之选，以上课无聊、任务量小、期末开卷考原题而著称。但给分评价非常混乱，有同学认为给分好（4.0），也有大量同学反映给分差（3.3），即使准备的材料相似，结果也可能天差地别。适合想划水且能接受绩点不确定性的同学。
    • 使用教材： 《思想道德与法治》

    李义天老师

    • 课程介绍： 老师研究哲学，讲课有趣，采用启发式教学，鼓励学生提问和回答，积极互动会有加分。
    • 课程任务： 任务量较大（“肝”）。包括慕课、读书报告（约2000字）、期末现场论文（约1500字）。核心任务是小组pre，占分多且有同学互评，竞争较激烈。有“微沙龙”形式可替代pre，较为轻松。
    • 评价： 课程有趣但硬核，想取得好成绩需要投入较多精力。pre是关键环节，比较“卷”。如果能选上“微沙龙”，课程会轻松很多。
    • 使用教材： 《思想道德与法治》

    中国近现代史纲要

    李蕉老师

    • 课程介绍： 被学生尊称为“蕉爷”。老师讲课水平极高，内容深刻，能帮学生打开回望历史的新视角。但与之对应的是高要求和高投入。
    • 课程任务： 课程核心是一次要求极高、竞争激烈的小组pre（读书展示），非常“肝”和“卷”。此外，每人还需完成一篇论文。
    • 评价： 大“卷”课，不适合想轻松过关的同学。小组成员的水平和投入度直接影响最终成绩，组内合作压力大。虽然过程辛苦，但收获巨大。给分据说呈正态分布，且依赖小组排名和组内互评，有一定风险。

    舒文老师

    • 课程介绍： 讲课风格比较平淡，甚至“催眠”。采用二维码签到，缺勤一次总评扣分较多（如10分），可以签到后离开。
    • 课程任务： 任务量少。主要为一篇读书报告和期末开卷考试（可带电脑，答案基本在PPT上）。被随机点到做课堂展示可替代读书报告。
    • 评价： “事少给分好”的代表。想拿高分的关键在于认真“卷”读书报告的字数。

    夏清老师

    • 课程介绍： 老师“人美心善”，讲课娓娓道来，水平很高，被同学评价为“真神”。PPT制作精良，讲解清晰。
    • 课程任务： 任务量少，无任何考试。
    • 评价： 给分极好。是一门广受好评的“必冲”课程，推荐。

    王宪明老师

    • 课程介绍： 讲课主要是念PPT，有时会引申跑题。
    • 课程任务： 课程形式变化巨大。以往是“一次小组pre + 期末开卷考”，任务量小。但近期（25春）大幅增加任务，变为“强制参加辅导课 + 抢pre名额 + 随堂小测 + 期末考试 + 3000字论文”。
    • 评价： 已从“事少分好”的推荐课“黑化”为任务繁重的课程。近期体验差，助教管理也存在问题，不推荐。

    欧阳军喜老师

    • 课程介绍： 老师被称为“反卷斗士”，讲课内容客观深刻。但上课会用雨课堂随机点名回答问题，需要保证出勤。
    • 课程任务： 任务量不大。包括一次人数众多的小组pre（可以划水）、一篇2000-5000字的读书报告和期末开卷考试。
    • 评价： 给分不错，但有同学反映可能比较随机（有高分也有2.7的个例）。想拿高分，建议在读书报告和考试中多写一些内容。

    李玉蓉老师

    • 课程介绍： 课堂需要“卷”回答问题以获得平时分，竞争激烈。有考勤。
    • 课程任务： 无pre。期中一篇3000字论文（需要查阅较多文献），期末为开卷/半开卷考试。
    • 评价： 任务量中等。期末考试需要整合书中多处信息，而非直接照搬，对答题速度和概括能力要求高。给分随机性较大，不建议写字慢或反感课堂“卷”回答的同学选择。

    李玓老师

    • 课程介绍： 老师人很好，会听取学生意见改进课程，推荐的书籍质量也很高。有考勤和随机点名。
    • 课程任务： 任务量不大。包括一次展示、500字期中作业和期末读书报告。
    • 评价： 推荐。投入时间不多也可获得不错的成绩（如A）。

    马克思主义基本原理

    邹广文老师

    • 课程介绍： 课程内容偏哲学，老师常提出一些新奇的观点，内容较为有趣。
    • 课程任务： 一次课堂展示、一篇课程论文以及一次开卷的期末考试。
    • 评价： 关于给分的评价不一，有同学认为给分很好，也有同学认为不理想。期末考试的题目基本出自PPT中的思考题，与课本内容关联不大。在回答开放性问题时，真诚地结合老师的观点和文献可能会有帮助。
    • 使用教材： 无

    艾四林老师

    • 课程介绍： 老师的讲授被评价为“高屋建瓴，如沐春风”。课程有时会由两位老师先后讲授（前半学期艾老师讲哲学，后半学期另一位老师讲历史）。
    • 课程任务： 两次小班讨论（30%）、一次课堂展示（10%）、课堂表现及小测（10%）和期末开卷考试（50%）。期末考试可上网查资料，小测主要用于考勤或提意见。
    • 评价： 强烈推荐，给分很好。但学期中不公布任何平时成绩，导致最终得分有些“莫名其妙”。正常完成各项任务、不“卷”也能获得不错的成绩（如A-）。
    • 使用教材： 马列主义经典著作导读，艾四林编

    王峰明老师

    • 课程介绍： 老师有许多个人独特的理论观点，上课常“diss”课本，课堂氛围有趣，同学间发言互怼是一大特色。
    • 课程任务： 基本没有平时作业和pre，主要考核为期末论文。
    • 评价： 强烈建议想认真听课、不摸鱼的同学选择。听课和记笔记至关重要，因为期末论文的成绩高度依赖于是否运用了老师上课讲的理论和原话。若笔记不全，可去知网搜索老师的论文作为参考。
    • 使用教材： 无

    李成旺老师

    • 课程介绍： 老师讲课深刻有趣，采用引导式教学。
    • 课程任务： 一次期中Pre，一次期末当堂半开卷论文。有小测用于点名。
    • 评价： 给分主要看期末论文的质量，建议提前准备时政热点相关资料。总体评价不错。
    • 使用教材： 无

    王贵贤老师

    • 课程介绍： 老师讲课认真，有助于加深对马原的理解。期末考核方式（考试或论文）由班级投票决定。有小测用于签到。
    • 课程任务： 一次小组pre（小组人数多，但竞争较“卷”）和期末考核（论文或开卷考试二选一）。
    • 评价： 整体给分不错。Pre部分是拿高分的关键，认真“卷”一下或在形式上创新（“整点花活”）效果很好，有小组全体拿A的先例。期末考试内容基本来自PPT，论文则看重选题的想象力。
    • 使用教材： 无

    朱安东老师

    • 课程介绍： 老师主攻政治经济学，授课内容也以此为核心。课程包含慕课、小班讨论等环节。
    • 课程任务： 任务量适中，包括慕课答题、两篇读书报告、三次小班讨论以及一篇重要的期末论文，无期末考试。
    • 评价： 推荐，给分很好。在讨论课上积极主动，论文写出自己的想法，容易获得好成绩。
    • 使用教材： 无

    龙治铭老师

    • 课程介绍： 老师讲课内容偏向政治经济学，对《资本论》有深入讲解，言之有物，并且很体谅学生。
    • 课程任务： 无小组任务，任务量适中。包括两道思考题（五选二）和一篇读书笔记（摘抄与个人内容比例1:1，不超过10000字）。
    • 评价： 给分非常不错，优秀率会给满。对于想深入理解政治经济学的同学是绝佳选择。
    • 使用教材： 无

    王代月老师

    • 课程介绍： 老师和蔼可亲，但授课内容偏学术化，可能不易听懂。上课会随机提问。
    • 课程任务： 近年任务量有所增加，包括：4-5次随堂小测、一篇5000字以内的论文（含初稿和终稿）以及一次pre。
    • 评价： 虽然任务量变大，但给分依然超好。有同学反映即使划水乱写也拿了A，说明给分标准可能比较宽松。总体上仍比较推荐。
    • 使用教材： 无

    刘恩至老师

    • 课程介绍： 老师讲课十分有趣，逻辑清晰，上课体验极佳。每节课开始会找同学回答问题。
    • 课程任务： 主要为期末开卷考试，写字量较大。
    • 评价： 给分很好。最重要的是认真听课并记好笔记，因为期末考试内容与课堂讲授高度相关。参考往年试题（重合度高）也是很好的备考策略。
    • 使用教材： 无

    毛泽东思想和中国特色社会主义理论体系概论（毛概） & 习近平新时代中国特色社会主义思想概论（习概）

    解安老师

    • 课程介绍： 课堂上会设置问答环节，助教记录回答次数。
    • 课程任务： 近期（2025春）考核方式为一次小组pre和期末随堂开卷论文。以往有过3-5k读书笔记的形式。
    • 评价： 给分总体很不错（A-或A），但有同学感觉略“玄学”。一学期内积极回答问题是取得高分的关键。

    陈明凡老师

    • 课程介绍： 讲课思路清晰，但会通过突击小测进行点名。
    • 课程任务： 任务量适中。主要包括一篇5000字读书报告、两次小班讨论和两次课堂突击小测。
    • 评价： 给分较“玄学”。正常完成所有任务，不参与课堂回答，成绩可能在A-到B+之间。积极在课堂或网络学堂讨论区回答问题，给分会更好。也有同学反映即使认真完成任务，最终得分也不理想。

    何建宇老师

    • 课程介绍： 老师人很nice，上课有趣，广受好评。
    • 课程任务： 任务形式每年都有所不同，但通常围绕“小班讨论+专题评述/报告+大作业”展开，例如2024年春季学期的任务为：考勤与课堂互动（20%）、小班讨论及专题评述（40%）、案例编写与分析（40%）。
    • 评价： 给分极好，工作量适中，是被普遍“冲就完事了”的推荐之选。

    冯务中老师

    • 课程介绍： 老师和善幽默，讲课水平被认为是思政老师中的前列。他会在寒假就公布学期的所有作业要求，方便学生提前规划。
    • 课程任务： 任务可以集中在假期完成，学期中会比较轻松。有平时分（10%）、小作业（30%）、大作业（60%）。大作业选题非常多样和新颖（如“诗词中的中国近现代史”、“家族史访谈”等）。
    • 评价： 给分不错（A-或A）。考勤非常重要，缺勤一次可能导致总评下降一个等级（如从A到B+）。

    郎昆老师

    • 课程介绍： 2025年春季新开课老师。上课内容有趣，富含个人见解，会邀请嘉宾讲座。老师非常为学生着想。
    • 课程任务： 任务量小。考核方式在第一节课由全体同学投票决定，通常为考勤和期末论文，无小组pre。
    • 评价： 强烈推荐。作业成绩会公布，过程透明。给分非常好。

    陈晓仪老师

    • 课程介绍： 2025年春季新开课老师，人非常善良。考勤方式多样，但大规模考勤会提前通知。
    • 课程任务： 无考试，纯写论文。10%考勤，30%小作业（1.5k字，三选一），60%大作业（2.5k字，二选一）。
    • 评价： 因为是纯论文考核，给分有些“玄学”，依赖于写作水平。

    车宗凯老师

    • 课程介绍： 【毛概】老师年轻，吐字清晰，讲课用心。早八课程，但基本不点名，代之以开课后五分钟内完成的雨课堂题目（答对与否不重要，完成一定数量即可满分）。
    • 课程任务： 两次随堂写作（各10%），一篇期中读书报告（50%），一篇期末课程总结（25%）。
    • 评价： 认真完成书面作业可获A-。课程安排清晰，体验不错。

    程正雨老师

    • 课程介绍： 【习概】老师发布考核标准较晚，引发学生不满。鼓励使用AI完成作业。
    • 课程任务： 有若干次课堂作业（可能用于签到），以及一次小组pre，pre投票高的组有加分。
    • 评价： 评价主要集中在课程安排上，给分情况不详。

    肖贵清老师

    • 课程介绍： 老师上课喜欢分享个人经历。
    • 课程任务： 一次小组展示和一篇3000字期末论文（老师明确表示1500-3000字即可，多写无益）。
    • 评价： 给分“一般”或“还可以”。

    李戈老师

    • 课程介绍： 会随机点名回答问题，因此需要到场听课。
    • 课程任务： 两篇小报告和一篇大报告，总字数约4000字。
    • 评价： 给分“一般”，但也有同学觉得“还可以”。

    路子达老师

    • 课程介绍： 无。
    • 课程任务： 无。
    • 评价： 【习概】给分差，有多个3.6和3.3的样本，不推荐。
